# PageSync

Keep currently open page between two devices in sync (for example, typing on laptop and handwriting on tablet).

## Usage

1. In plugin settings, set one device to `Host` and others to `Client`.
2. Press `Sync` on host and client devices in the left sidebar (or type "/sync page" command).
